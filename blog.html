Some interesting Javascript related question!

01. What is DOM and what is the purpose of DOM?

DOM means Document Object Model. All HTML webpages are reffered to as documents. The Document Object Model represents each of this webpages in a tree like structure to make it easier to access and manage the elements.

02. How will you select HTML elements using DOM? Name the DOM methods.

There are several ways to select HTML elements-
01. document.querySelector()
02. document.querySelectorAll()
03. document.getElementById()
04. document.getElementsByClassName()
05. document.getElementsByTagName()
06. document.forms[].elements[]

03. What is event bubble?

Event bubble is a important concept of DOM. It is one of the ways of event propagation in the HTML DOM. Event bubble propagate the DOM tree upwards.

04. What is a callback function and why will you use it?

In JavaScript, callback function is a function passed into another function as an argument.
